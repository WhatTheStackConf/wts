# Content

Use this when changing Markdown, MDX, Velite configuration, or content-driven routes.

- Content files live under `content/`.
- Velite is configured in `velite.config.ts`.
- Import generated content from `.velite` or through existing content helpers.
- `pnpm dev` runs `velite --watch` through `pnpm start:dev`.
- `pnpm build` runs `velite` before the Vite production build.
